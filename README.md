Acest proiect include identificarea sistemelor de ordinul doi/superior folosind metode parametrice si neparametrice.
Proiectul 1-idetificarea unui siatem de ordinul doi cu poli reali folosind raspunsul la treapta.
Proiectul 2-identificarea aceluiasi sistem ca si la proiectul1,doar ca identificare pe un semnal de tip chirp
Proiectul 3-Identificarea unui sistem folosind metode parametrice validate prin intercorealtie si intercorelatie folosind un semnal de tip spab
